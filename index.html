Arranque seguro en Linux

Antes de empezar debemos configurar las particiones en la siguiente imagen pongo como se deberia poner cada partición:

Le damos ha hecho, luego nos pedira los tipicos datos de registros los rellenamos y le damos ha instalar.

Esperamos aprox 2 min y ya tendriamos la maquina lista.

En la siguiente seccion vamos ha realizar la Procteccion de BIOS/UEFI.

Ejecutamos este comando para el arrance con contraseña

nano /etc/default/grub

Le damos enter y te saldra esta ventana:

Aqui modificados el GRUB_TIME_OUT=5 y quitamos el 5 y ponemos 0

Guardamos cambios con Ctrl-O, y salimos del editor de nano con Ctrl-X.

Recuerda que es necesario regenerar fichero para guardar cambios con este comando:

sudo update-grub

Para proteger el arranque con contraseña debemos ejecutar este comando:

sudo nano /etc/grub.d/40_custom

Cuando le damos al enter nos aparecera un editor diferente:

Agregamos al final set superusers="root" y password root CONTRASEÑA. Recuerda que es necesario regenerar fichero para guardar cambios:

En este caso de user "root" y de password "1234"

sudo update-grub

Aqui se finalizaria el arranque seguro de Linux.
